//EtabsAssistance
EtabsAssistance is a standalone desktop application designed to streamline and automate structural engineering workflows within CSI ETABS. 
By combining a user-friendly graphical interface with powerful computational modules and direct API integration, this tool minimizes manual effort and improves efficiency in structural modeling.

Key Features
Computational Modules:

Static Seismic Analysis: Easily perform seismic calculations based on structural parameters.

Dynamic Analysis Configuration: Simplify the setup and fine-tuning of dynamic analysis parameters.

Parameter Definition: A streamlined dashboard to input design values and reduce calculation errors.

ETABS API Integration:

Automated Load Combinations: Directly interface with the CSI ETABS API to generate and export complex load combinations into your model automatically, saving significant time.

Installation & Usage
No Python knowledge or environment setup is required.

EtabsAssistance is provided as a standalone Windows executable.

Head over to the Releases section.
Download the latest installer (.exe or .msi).
Run the installer and follow the on-screen instructions to get started immediately.
